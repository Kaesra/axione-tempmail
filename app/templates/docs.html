DNS + MX

Trafik akisi nasil olmali?

Web trafik

`mail.axione.xyz` -> Cloudflare Tunnel veya reverse proxy -> `127.0.0.1:{{ web_port }}`

Mail trafik

`MX` -> `mx1.axione.xyz` -> sunucu public IP -> `{{ smtp_port }}/tcp` SMTP listener

DNS Blueprint 
mail    CNAME   <cloudflare-tunnel-host>   proxied
mx1     A       <server-public-ip>        dns-only
@       MX      mx1.axione.xyz            priority 10
@       TXT     v=spf1 mx -all
_dmarc  TXT     v=DMARC1; p=none; adkim=s; aspf=s

Environment

.env icinde dikkat etmen gereken alanlar

TEMPMAIL_DB_URL=sqlite:///./tempmail.db
TEMPMAIL_WEB_HOST=0.0.0.0
TEMPMAIL_WEB_PORT={{ web_port }}
TEMPMAIL_SMTP_HOST=0.0.0.0
TEMPMAIL_SMTP_PORT={{ smtp_port }}
TEMPMAIL_ACCEPTED_DOMAINS={{ accepted_domains[0] if accepted_domains else 'axione.xyz' }}
TEMPMAIL_SESSION_HOURS=72
TEMPMAIL_SECURE_COOKIES=false
TEMPMAIL_ADMIN_USERNAME=admin
TEMPMAIL_ADMIN_PASSWORD=change-me-now

Production notu

HTTPS aktifse `TEMPMAIL_SECURE_COOKIES=true` yap. Varsayilan admin sifresini production'a tasima.

Mail notu

Dis dunyadan mail alacaksan `TEMPMAIL_SMTP_PORT=25` neredeyse her zaman gerekli olur.

Production Checklist

Canliya cikmadan once son kontrol

Inbox'a Don

1. DNS hazir

`mail`, `mx1`, `MX`, `SPF`, `DMARC` kayitlari dogru.

2. SMTP erisimi var

Sunucuya disaridan `{{ smtp_port }}/tcp` erisilebiliyor.

3. App ayakta

FastAPI ve SMTP servisi systemd veya baska bir manager ile ayakta.

4. Secrets guvenli

Admin sifresi degistirilmis, secret dosyalari repoya cikmiyor.
